# Multisales
pagina web  de e-commerce llamada MultiSales. 
pagina web  de e-commerce llamada MultiSales. La web debe tener:
•	Un catálogo público con búsqueda de productos, tarjetas con imagen, nombre, descripción y precio, y la opción de seleccionar múltiples productos para añadirlos al carrito de una sola vez.
•	Un carrito de compras persistente con un drawer lateral que muestre los productos, cantidades, total y un botón para ir al checkout.
•	Una página de checkout que muestre el resumen del pedido y permita confirmar la compra.
•	Un panel de administración (solo visible para admins) donde se puedan crear, editar y eliminar productos, subir imágenes y hacer eliminaciones masivas.
•	Layout con navegación que incluya el logo, acceso al admin y el ícono del carrito con contador de items.
•	Todo en español, con diseño moderno, minimalista y responsivo (mobile + desktop)."**
•	Botón estadística que contenga un botón de estadística que me muestre el volumen de ventas,  total diario mensual y anual para llevar mi estadística que cuando entre me muestre una gráfica de áreas y de líneas con sombra tenue incluya una tarjeta que me muestre el valor diario mensual y  anual cada tarjeta debe mostrar un número más pequeño la comparación día mes año anteriores a la actual y se coloque verde cuando se haya superado el periodo anterior según sea el caso y permanezca rojo cuando no lo haya hecho. este boto se deme ingresar por el home casita al lado de nombre del canal.
